# Product Requirements Document (PRD)

## Enigma Connect Unified Website

### 1. Overview

Enigma Connect Unified Website adalah penggabungan dari tiga landing page produk:

* Enigma Connect
* Enigma TMS (Transport Management System)
* Enigma WMS (Warehouse Management System)

Website ini akan menjadi **portal utama** yang memperkenalkan ekosistem produk Enigma kepada pengguna logistik di Indonesia.

Target utama website ini adalah:

* Perusahaan logistik
* Operator transportasi
* Manajer gudang
* Distributor
* Perusahaan manufaktur

Website harus **mudah dipahami**, **cepat diakses**, dan **jelas menjelaskan manfaat produk**.

---

# 2. Goals

Tujuan utama dari website ini:

1. Menyatukan tiga landing page menjadi satu website utama.
2. Memperkenalkan ekosistem produk Enigma kepada calon pengguna.
3. Memudahkan pengguna untuk memahami manfaat TMS dan WMS.
4. Menyediakan halaman login untuk masing-masing sistem.
5. Menyediakan halaman signup untuk calon pelanggan.
6. Meningkatkan konversi demo dan pendaftaran produk.

---

# 3. Target Users

## Primary Users

1. Logistics Manager
2. Warehouse Manager

4. Fleet Manager

## Secondary Users

1. Business Owner

3. Supply Chain Manager

# 4. Key Value Proposition

Enigma Connect menyediakan solusi digital untuk membantu perusahaan logistik:

# 6. User Flow

## New Visitor Flow

User membuka website → melihat informasi produk → klik "Pelajari Produk" → melihat detail produk → klik "Daftar" atau "Hubungi Kami".

---

## Login Flow

User membuka halaman login → memilih sistem → diarahkan ke halaman login sistem.

Example:

/login?tms
/login?wms

---

## Signup Flow

User membuka halaman signup → mengisi form → submit → data dikirim ke backend / CRM.

# 10. SEO Strategy

Website Enigma Connect harus dioptimalkan untuk mesin pencari agar mudah ditemukan oleh perusahaan logistik di Indonesia.

Tujuan SEO:

* meningkatkan traffic organik
* meningkatkan permintaan demo
* meningkatkan signup produk

Target utama pencarian adalah kata kunci yang berhubungan dengan:

* sistem logistik
* manajemen transportasi
* manajemen gudang
* software TMS
* software WMS

---

# 10.1 Target Keywords

Primary keywords:

* software TMS
* sistem manajemen transportasi
* software WMS
* sistem manajemen gudang
* software logistik

Secondary keywords:

* aplikasi transport management system
* aplikasi warehouse management system
* software logistik Indonesia
* sistem manajemen armada
* software distribusi barang

---

# 10.2 SEO Landing Pages

Website harus memiliki halaman khusus untuk setiap produk agar SEO lebih optimal.

Halaman SEO utama:

* / → Enigma Connect
* /produk/tms → Enigma TMS
* /produk/wms → Enigma WMS

Setiap halaman harus memiliki:

* judul halaman yang jelas
* deskripsi produk
* fitur utama
* manfaat produk
* CTA (Daftar / Demo)

---

# 10.3 Content Strategy

Konten harus menggunakan bahasa Indonesia yang mudah dipahami oleh pengguna logistik.

Pedoman penulisan:

* gunakan kalimat pendek
* gunakan bullet points
* gunakan heading yang jelas
* hindari istilah teknis yang rumit

Contoh:

❌ Sistem optimalisasi distribusi berbasis algoritma.

✔ Atur rute pengiriman agar lebih cepat dan hemat biaya.

---
