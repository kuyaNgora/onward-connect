# Enigma Connect - Unified Website

Website unifikasi tiga produk logistik Enigma: Enigma Connect, Enigma TMS (Transport Management System), dan Enigma WMS (Warehouse Management System).

## 📋 Overview

Enigma Connect Unified Website adalah portal utama yang memperkenalkan ekosistem produk Enigma kepada pengguna logistik di Indonesia. Website ini menggabungkan tiga landing page produk menjadi satu platform terpadu.

## 🎯 Target Users

- **Perusahaan Logistik** - Mengelola rantai pasok dan distribusi
- **Operator Transportasi** - Mengawasi armada dan pengiriman
- **Manajer Gudang** - Mengatur inventory dan operasional gudang
- **Distributor** - Mengelola distribusi barang ke berbagai lokasi
- **Perusahaan Manufaktur** - Mengatur supply chain dan logistik

## 🚀 Tech Stack

- **Framework**: React 18.3.1 with TypeScript
- **Build Tool**: Vite 5.4.2
- **State Management**: Redux Toolkit
- **Routing**: React Router DOM
- **Styling**: Tailwind CSS v4.2.2
- **SEO**: React Helmet Async

## 🛠️ Installation

```bash

git clone <repository-url>
cd enigma-connect

# Install dependencies
npm install
```

## 🔧 Development

```bash
# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview

# Lint code
npm run lint
```
